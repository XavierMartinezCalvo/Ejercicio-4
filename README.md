# ConversorDec2Bin
## Explicación del código
1. Abrir el archivo dec2bin.py
2. El programa te pide que añadas un número decimal para pasarlo a binario
3. El programa ahora te pide el número bits que quieres para convertirlo
4. El programa ejecuta la conversión
5. Muestra por pantalla el resultado de la conversión
